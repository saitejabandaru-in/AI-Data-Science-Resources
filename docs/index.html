💡 The Dog Trick Analogy

A single neuron is like training a puppy. Every time it predicts correctly (points on the right side of the line), we leave weights alone. When it makes an error, we gently pull the leash (adjust the weights) in the direction of the correction, scaled by our learning rate.

📊 Scientific Computing Performance Benchmarks

Measure and visualize the computational efficiency of data manipulation engines. This lab charts the exact execution times for common operations (Group By, Filter, Join) across 1 Million Rows comparing Python loops, Pandas, and Rust-based Polars.

🐍 Pure Python Loops

  • Execution Time: ~2,400 ms
  • Speed Index: 1x (Base)
  • Implication: Single-threaded, bound by the GIL. Heavy CPU overhead for loops.

🐼 Pandas (C-Optimized Vectors)

  • Execution Time: ~120 ms
  • Speed Index: 20x Faster
  • Implication: Columnar numpy memory structures. Bypasses Python loops, but still single-threaded and eager.

🦀 Polars (Rust Parallel Engine)

  • Execution Time: ~8 ms
  • Speed Index: 300x Faster
  • Implication: Multi-threaded execution, lazy query plan optimization, and native Rust speed.

Want to run these benchmarks locally with your actual hardware performance? Execute the script in your local workspace:

python3 scripts/benchmark_lab.py
